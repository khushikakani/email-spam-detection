# Email Spam Detection

## Overview

This project implements an Email Spam Detection system using Machine Learning techniques. The model analyzes email messages and classifies them as Spam or Ham (Not Spam).

## Objective

To build a predictive model that can identify unwanted spam emails and improve email filtering.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## Project Structure

* Email_Spam.ipynb : Main notebook containing data preprocessing, model training, and evaluation.

## Features

* Data cleaning and preprocessing
* Text feature extraction
* Spam/Ham classification
* Model performance evaluation

## How to Run

1. Clone the repository.
2. Open `Email_Spam.ipynb` in Jupyter Notebook.
3. Install the required libraries.
4. Run all cells to train and test the model.
